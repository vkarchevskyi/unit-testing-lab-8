# Лабораторна робота № 8 з предмету "Проектування інформаційних систем"

## Тема дипломної роботи: «Розробка вебдодатку з елементами ШІ для формування розкладу занять»

Був зроблений прототип з кількома сутностями, які взаємодіють між собою.

* Група
* Викладач
* Предмет
* День тижня

На основі цих сутностей створено клас ScheduleValidator, який має 3 методи:

1. validateLesson - перевіряє коректність одного запису пари
2. detectConflicts - виявляє конфлікти між записами пар
3. calculateLoad - рахує навантаження викладача

Усі три методи були протестовані у файлі `schedule-validator.test.ts`.

Для запуску тестів необхідно встановити Bun та виконати команду:

```bash
bun test
```
